Mini-processeur RISC-V 32 bits
Projet réalisé dans le cadre du cours d'Architecture des ordinateurs à l'Université de Perpignan.
Description
Conception et simulation d'un mini-processeur RISC-V 32 bits à l'aide du logiciel Digital. Le projet couvre l'ensemble des composants bas niveau d'un processeur ainsi que le cycle d'exécution complet des instructions.
Composants implémentés

UAL 32 bits — opérations arithmétiques et logiques (addition, AND, OR...)
Registres — implémentés avec des bascules D
Circuit de contrôle — gestion des signaux de contrôle via table de vérité
Compteur programme (PC) — suivi de l'instruction courante
RAM — mémoire de données
Instructions de branchement (BR) — gestion des sauts conditionnels
